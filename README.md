# Address Book – JSON Server

This is a simple and modern Address Book web application built using HTML, CSS, and JavaScript with JSON Server as a mock backend.

The app allows users to add, edit, delete, search, and sort contacts through a clean card-based interface. It uses a modal form for input and shows toast notifications for actions like save and delete.

## Features
- Add / Edit / Delete contacts
- Real-time search
- A–Z / Z–A sorting
- Contact count display
- Modern card UI
- JSON Server as backend

## Tech Stack
HTML, CSS, JavaScript, JSON Server

## How to Run

Install dependencies:

npm install

Start JSON Server:

npm start

Then open `index.html` in your browser.

This project was created to practice CRUD operations, API integration, and building a clean, modern UI using vanilla JavaScript.
